Introducción

Este programa es una calculadora de integrales definidas la cual retorna el area de una expresión dada. Este programa está realizado en JavaScript clásico
Tomando como Parametros:

  • Límites ingresados superior e inferior
  • Tipo de expresión f(x)
  • El tamaño de la iteración

Límites

Los límites soportan el ingreso tanto de operaciones matemáticas como de grados y números reales.
Alguna de las entradas que soporta son:

  • 3 o -3
  • 90°
  • 3*pi/2

Expresiones

Las expresiones que soporta son:

  • 3 o -3
  • 90°
  • 3*pi/2

Tamaño de Iteración

El tamaño de la iteración es el avance en el eje x con respecto al cálculo utilizado para ejecutar una función de evaluación con la respectiva expresión dada f(x) culaquiera sea.

Uso

Las integrales definidas también se utilizan en probabilidad, administración, economía, ecología, computación, arquitectura, en las ingenierías (civil, eléctrica, mecánica, etc.)
y en muchas otras ramas de las ciencias. Algunos resultados importantes en ingeniería se demuestran con el uso de las integrales definidas.

Ejemplos

A continuación se muestran algunos ejemplos de cómo se ve la página web y cómo se pueden utilizar sus características:

  • Cálculo de área: Los usuarios pueden buscar recetas por ingrediente o tipo de comida.

  • Calificar recetas: Los usuarios pueden calificar las recetas de otros usuarios y ver las calificaciones de las recetas.
